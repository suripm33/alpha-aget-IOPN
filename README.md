# Alpha Agent

Alpha Agent is an AI assistant for DeFi builders and users to research OPN Chain protocols, prepare swap, locker, and perp workflows, and understand on-chain finance tasks.

Built for the OPN Builders Programme — Season 1: DeFi & Open Finance.

## Problem

DeFi can be confusing for new users. Before interacting on-chain, users often need to understand wallets, networks, swaps, liquidity, approvals, lockers, perps, contract addresses, and transaction risks.

Alpha Agent is designed to simplify that process with clear step-by-step guidance.

## What Alpha Agent does

- Helps users research OPN Chain DeFi protocols
- Prepares swap workflow checklists
- Explains locker and liquidity actions
- Helps users understand perp and open finance workflows
- Gives beginner-friendly guidance before users interact on-chain
- Provides project and contract context for safer DeFi exploration

## Season 1 focus

The current focus is DeFi & Open Finance on OPN Chain.

Alpha Agent is positioned as an assistant layer for users who want to explore OPN ecosystem opportunities, understand DeFi tasks, and prepare actions before using protocols such as swaps, lockers, liquidity tools, and perp-style applications.
